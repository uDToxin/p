# Mini Cloud VPS

## Backend
cd backend
pip3 install -r requirements.txt
python3 app.py

## Frontend
cd frontend
npm install
npm start

## Telegram Bot
cd telegram_bot
pip3 install -r requirements.txt
python3 bot.py

Access Web Panel at http://localhost:3000
Telegram Bot commands: /create_vps /list_vps /delete_vps
